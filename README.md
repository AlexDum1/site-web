# site-web

Le site vitrine DumAlgo est dans `DumAlgo/` (`index.html` + `assets/`).
Le téléphone, l'email et l'endpoint du formulaire sont centralisés dans `DumAlgo/assets/js/main.js` (objet `SITE`) — à compléter avant mise en ligne, ainsi que les vraies captures d'écran dans `DumAlgo/assets/img/`.

## Maquettes prospects

Maquettes one-page de démonstration pour prospects (prospection artisans locaux, Le Pouget).
Chaque dossier = une maquette pour un prospect (slug = nom de l'entreprise + commune).

Ces pages sont des maquettes de démonstration, non indexées (noindex), montrées à titre d'exemple avant tout accord du prospect.
